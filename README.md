# ✈️ YouTube Travel Insight Pro

Une application web intelligente qui analyse les commentaires YouTube pour prédire l'intention de voyage et analyser le sentiment de l'audience.

## ✨ Fonctionnalités
- **Scraping Automatisé** : Récupère les commentaires via Selenium.
- **Analyse de Sentiment** : Détermine si l'audience est enthousiaste.
- **Machine Learning** : Prédit la probabilité de visite d'un pays.
- **Visualisation interactive** : Graphiques Plotly et détection automatique du drapeau du pays.
